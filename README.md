## Installation & Usage
```
  git clone https://github.com/sleepychloe/matrix.git
  cd matrix
  bash ./test.sh [arg1] [arg2]
```
[arg1]: make, fclean, re, run<br>
[arg2]: (make, fclean, re) all, 00 to 15<br>
&nbsp;&nbsp;(run) 00 to 15

example
```
  bash ./test.sh make all
  bash ./test.sh run 00
  bash ./test.sh fclean all
```
